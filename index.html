Les règles du ROTUS

Comme dans le jeu Motus, vous devez trouver le mot en n'ayant que la première lettre

Une lettre avec un fond vert signifie que la lettre est bien placé dans le mot

Une lettre avec un fond bleu signifie que la lettre est mal placé dans le mot

Si le fond ne change pas, la lettre n'existe pas dans le mot
